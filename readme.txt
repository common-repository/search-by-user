Search by user adds an author filter to the post admin page.

== Description ==

Search by user adds an author filter to the post admin page.  The drop down list includes all registered users for the blog.

== Installation ==

1. Upload `search_by_user.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
